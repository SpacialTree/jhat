Installation

jhat works on Python 3+ and requires the following Python packages:

  • numpy

  • matplotlib

  • astropy

  • jwst

  • scipy

  • photutils

  • pysiaf

  • astroquery

  • pandas

  • stsci skypac

Install using pip

Using pip:

pip install jhat

Install Development Version

Using git:

git clone https://github.com/arminrest/jhat.git
cd jhat
python setup.py install
